
# RS-FROST
**RS-FROST** is a rust UI and multimedia library.<BR>

It is a rust version of **[Frost](https://github.com/djordjermus/frost-old/)** and **[Cauldron](https://github.com/djordjermus/cauldron/)**<BR>
